Players go head to head in this turn based board game that plays like a mix between chess and tic-tac-toe. The game is currently available to play on kongregate and will soon be accessible through the chrome store. There are plans in the near future to release the game on mobile phones.

The game runs on HTML5 inside the browser. Client side code leverages AngularJS, CoffeeScript and Less and is delivered to the client via Amazon's CloudFront CDN.

The servers run NodeJS with Socket.io as the communication layer and are hosted on Amazon EC2. The nodes hit a MongoDB hosted by MongoHQ for persistant storage and a Redis server (also EC2) for everything else.
