Overview

Tile Titans is an HTML5 game that I developed as a side project over the summer of 2013. It is a turn based board game that plays like a mix between chess and tic-tac-toe.

Skills

  • AngularJS
  • CoffeeScript
  • LESS
  • NodeJS
  • Socket.io
  • MongoDB
  • Redis
  • Amazon EC2

Key Learnings

When developing Tile Titans I spent too much time developing non-gameplay features such as a store for purchasing additional tiles and a way to add and challenge friends. I should have only focussed on developing the the core game and a simple matchmaking system and then developed the other features after launching it on Kongregate.

Project Motivation

I developed a V1 of the game as part of the UKEN games hackathon

Tech Stack

The game runs on HTML5 inside the browser. Client side code leverages AngularJS, CoffeeScript and Less and is delivered to the client via Amazon's CloudFront CDN.
The servers run NodeJS with Socket.io as the communication layer and are hosted on Amazon EC2. The nodes hit a MongoDB hosted by MongoHQ for persistent storage and a Redis server (also EC2) for everything else.
